Урок 6: Введение в основы JavaScript

Работа с prompt(), if и else.

Задание 1: запросить информацию с помощью prompt() и сопоставить с имеющимися данными.

Код:


      let existingUserLogin = "the_best_user";
      let existingUserPassword = "12354678";
      let userLogin = prompt("Введите логин").trim();
      let userPassword = prompt("Введите пароль").trim();

      if (userLogin === existingUserLogin && userPassword === existingUserPassword) {
        alert(`Добро пожаловать, ${userLogin}!`);
      } else {
        alert("Логин и (или) Пароль введены неверно!");
      }
