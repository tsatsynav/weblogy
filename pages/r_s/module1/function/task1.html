Модуль 1. База JavaScript

Урок 8: Функции

Работа с function declaration, function expression, arrow function expression.

Задание 1: объявить функции разными способами.

Код:


      // function declaration
      function getName1(name) {
        return name;
      }

      console.log(getName1('Andrey'));

      // function expression
      const getName2 = function(name) {
        return name;
      };

      console.log(getName2('Dodo'));

      // arrow function expression
      const getName3 = (name) => {
        return name;
      }

      console.log(getName3('Milena'));
