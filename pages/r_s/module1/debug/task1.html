Урок 7: Debug

Работа с console.log() для Debug.

Задание 1: Исправьте все ошибки, чтобы программа работала правильно. Также выполните следующие условия: баг должен быть исправлен при помощи console.log() либо debugger, в коде должен присутствовать минимум один console.log() либо debugger.

Код:


      let health = Number(prompt('Введите число параметра "здоровье" для персонажа').trim());
      console.log(health);
      if (health < 0 || !health) {
        alert('Параметр "здоровье" должен быть больше нуля!');
        console.log('no success')
      } else {
        alert(`Параметр "здоровье" равен ${health}`);
        console.log('success')
      }
