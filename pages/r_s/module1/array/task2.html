Урок 9: Массивы

Работа с function declaration, unshift(), for.

Задание 2: Вам необходимо создать функцию getSumOfSequence(), которая будет создавать массив из чисел и высчитывать сумму первого и последнего элементов массива.

Функция принимает в себя один параметр number. Используя этот параметр, нужно создать массив из чисел от 1 до number. Для заполнения массива элементами используйте цикл for (например, если был передан number равный 5, то массив должен выглядеть следующим образом: [1, 2, 3, 4, 5]).

Код:


      function getSumOfSequence(num) {
        let array = [];
        for(let i = num; i > 0; i -= 1) {
          array.unshift(i);
          //console.log(array);
        }
      
        return array[0] + array[array.length - 1];
      
      }
      
      console.log(getSumOfSequence(10));
