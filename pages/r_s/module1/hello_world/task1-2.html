Урок 1: Hello World

Работа с console.log() и alert().

Задание 1: создание базовой разметки страницы и подключение файла с расширением .js.

Задание 2: Выведти в консоль строчку “Запускаем таймер” с помощью функции console.log(). Затем добавьте 6 функций alert(), каждая из которых будет выводить число, начиная с 5-ти и заканчивая нулем. После вывода нуля напишите строчку кода, которая будет выводить в консоль “Обратный отсчет закончен”.

Код:


      console.log('Запускаем таймер'); // console.log() - метод вывода сообщения в консоль браузера 
      alert('5'); // alert() - метод вывода модального окна с сообщением в браузере
      alert('4');
      alert('3');
      alert('2');
      alert('1');
      alert('0');
      console.log('Обратный отсчет закончен');
